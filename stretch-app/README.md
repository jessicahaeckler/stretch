# Workout Planner

A workout planning application built with Next.js App Router.

This project started by following the official Next.js App Router tutorial to learn modern Next.js patterns. It has since been adapted toward a workout planning application and is actively being expanded.

## Current features

- Dashboard
- Workout data
- PostgreSQL database
- Authentication (in progress)
- CRUD operations (in progress)

## Planned features

- Exercise library
- Workout templates
- Weekly scheduling
- User accounts
- Progress tracking
